# Sales Analytics Dashboard

## Project Overview
This project analysis sales data using Microsoft Excel and Power BI to uncover revenue trends, Average Order Value, Total Orders received across months.

## Tools Used
- Microsoft Excel
- Power BI

## Key Features
- Interactive dashboard
- Revenue Generated
- Average Order Value
- Quantities Sold
- Total Orders received
- Slicers and filters

## Key Performace Indicators
- Revenue Generated
- Orders received
- Quantities sold
- AOV

## Business Insights
This dashboard helps in decision-making for businesses, provides vlauable insights into product performance across countries and profit margin
