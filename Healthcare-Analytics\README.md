# Healthcare Analytics Dashboard

## Project Overview
This project analyzes healthcare data using Microsoft Excel and Power BI to uncover trends in patient demographics, hospital operations, financial performance and clinical outcomes

## Tools Used
- Microsoft Excel
- Power BI

## Key Features
- Interactive Dashboard
- Patient Demographics analysis
- Hospital Performance analysis
- Financial analysis
- Clinical Insights
- Dynamic filters and slicers

## Key Performance Indicators
- Total Patients
- Total Revenue
- Average Length of Stay
- Admission type

## Business Insights
This dashboard helps healthcare administrators identify operational trends, monitor financial performance, 
and support data-driven decision-making
